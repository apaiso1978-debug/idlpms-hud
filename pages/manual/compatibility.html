ปัญหา CORS คืออะไร?

CORS (Cross-Origin Resource Sharing) คือระบบความปลอดภัยของเว็บเบราว์เซอร์ที่ป้องกันไม่ให้เว็บไซต์หนึ่งแอบไปดึงข้อมูลจากอีกเว็บไซต์หนึ่ง ในกรณีนี้ Server ของ DLTV มีการตั้งค่าไว้ว่า "อนุญาตให้แสดงผลเฉพาะบนเว็บ dltv.ac.th เท่านั้น"

เมื่อคุณรันโปรเจกต์บน localhost เบราว์เซอร์จึงบล็อกการโหลดวิดีโอเพื่อความปลอดภัยตามมาตรฐานสากล
