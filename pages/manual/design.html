กฎเหล็ก (The Iron Rules)

  • • 14px Universal Standard: ขนาดเดียวทั้งระบบคือ 14px (text-[13px]) เท่านั้น (ห้ามเล็กกว่าหรือใหญ่กว่าสำหรับ Body)
  • • 45% Secondary Luminance: เมนูย่อยและข้อความรองต้องใช้ความสว่างระดับ 45% เท่านั้น
  • • 3px Corner Radius: บังคับใช้ `rounded-[var(--vs-radius)]` ทุกจุด (Unity Standard)
  • • 850-Baseline HUD: พื้นหลังหลักต้องใช้ค่าสี Zinc-850 Baseline (var(--vs-bg-deep)) เป็นมาตรฐาน
  • • Zero Glow Elevation: ห้ามใช้ Drop Shadow, Blur หรือ Glow ทุกจุด (100% Flat)
  • • 300px Sidebar: ความกว้างของ Sidebar คงที่ 300px เพื่อความสมมาตรทางสายตา
  • • 0% Unicode Policy: ห้ามใช้ไอคอนจาก Unicode ให้ใช้ CSS Mask เท่านั้น
  • • 1px Icon Precision: ไอคอนทุกตัวต้องมี `stroke-width="1"` เพื่อความคมชัดสูงสุด
  • • 3px Precision Bars: Progress bar และ Scroll bar ต้องมีความหนา 3px เพื่อความประณีต
  • • Unified Spacing Scale: ใช้ระบบ Padding เป็น 4-8-16-24px เท่านั้น (p-1, p-2, p-4, p-6) ห้ามใช้ p-8, p-12, p-10
  • • 24px Page Padding: ทุกหน้าต้องใช้ padding 24px (p-6) สำหรับ body element
  • • 16px Card Padding: ทุก Card และ Section ต้องใช้ padding 16px (p-4) ภายใน
  • • 1px Universal Border: เส้นทุกเส้น ขอบทุกขอบทุก Element บังคับใช้ความหนาแค่ 1px เท่านั้นทั้งระบบ
  • • 16px Gap Standard: ใช้ gap-2 (16px) สำหรับ grid/flex layouts ห้ามใช้ gap-6, gap-8

สุนทรียภาพระดับพรีเมียม (Premium HUD Aesthetics)

  • • Zero Fung: ห้ามใช้เทคนิคสีฟุ้งหรือความพร่ามัว (No Glow / No Blur)
  • • Industrial Typography: ความหนาของตัวอักษรต้องเป็น 200 (Extra Light) ทั้งระบบทุกจุด ห้ามใช้ 300 หรือ 500
  • • Heroicon CSS Mask: ใช้ระบบครอบหน้ากากไอคอนด้วย CSS เพื่อการเปลี่ยนสี

Structural Elements

  • • .dna-grid-line: เส้นหกเหลี่ยมซ้อนกัน 5 ชั้น stroke-dasharray: 4 2
  • • .dna-axis: 3 เส้นเต็มความยาว (K-F, P-L, A-M) Opacity 30%

03 Automated Enforcement

Pre-commit Hook Protection

ระบบติดตั้ง Pre-commit Hook อัตโนมัติ ที่จะตรวจสอบ Iron Rules ก่อนอนุญาตให้ commit ทุกครั้ง

text-[13px] — ห้ามใช้สำหรับภาษาอังกฤษ (ใช้ text-[13px] แทน)
bg-[var(...)]/opacity — ห้ามใช้ opacity shorthand (ใช้ rgba() แทน)
font-medium — ควรใช้ font-light (ยกเว้น hud-badge-micro)
tracking-* — ห้ามใช้ letter-spacing (ใช้ค่า default เท่านั้น)
