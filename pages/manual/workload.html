Teacher Workload & Smart Config

System Documentation

1. Legal Framework — กคศ. ว9/2564

สำนักงาน กคศ. กำหนดภาระงานขั้นต่ำของข้าราชการครู ตามหนังสือ ว9/2564 และ ว21/2564 ครูต้องมีชั่วโมงปฏิบัติงานรวม ไม่ต่ำกว่า 20 ชั่วโมงต่อสัปดาห์ โดยชั่วโมงสอนตามตารางสอน ไม่ต่ำกว่า 12 ชั่วโมงต่อสัปดาห์

Total Workload
≥ 20 ชม./สัปดาห์
Teaching Hours
≥ 12 ชม./สัปดาห์
Support & Others
≥ 8 ชม./สัปดาห์
PLC / Year
≥ 50 ชม./ปี

2. Four Domains of Teacher Workload

1
Hard Rule — กคศ.

ชั่วโมงสอนตามตารางสอน

คาบสอนในรายวิชา/กลุ่มสาระการเรียนรู้ตามหลักสูตร กิจกรรมพัฒนาผู้เรียน ลูกเสือ/เนตรนารี ชุมนุม และกิจกรรมรักการอ่าน

ขั้นต่ำ ≥ 12 ชม./สัปดาห์ — ลดไม่ได้แม้มีภาระงานพิเศษ
2
Soft Default

งานส่งเสริมและสนับสนุนการจัดการเรียนรู้

  • PLC (ชุมชนการเรียนรู้ทางวิชาชีพ)
  • การจัดทำแผนการจัดการเรียนรู้
  • การวัดและประเมินผล / สร้างสื่อ
  • งานครูที่ปรึกษา / ดูแลนักเรียน / โฮมรูม
3
Soft Default

งานพัฒนาคุณภาพการศึกษาของสถานศึกษา

  • หัวหน้ากลุ่มสาระการเรียนรู้
  • หัวหน้าสายชั้น
  • หัวหน้าฝ่าย (งบประมาณ / บุคคล / วิชาการ / ทั่วไป)
  • งานประกันคุณภาพการศึกษา (QA)
4
Soft Default

งานตอบสนองนโยบายและจุดเน้น

  • โครงการตามนโยบายรัฐบาล
  • กิจกรรมกระทรวงศึกษาธิการ
  • กิจกรรมสวดมนต์ / จิตอาสา
  • โครงการพิเศษของเขตพื้นที่

3. Design Pattern — Smart Default + Manual Override

ทุกค่าที่ configurable มี Default ที่ฉลาด (ใช้ได้เลย) + ผอ. สามารถ toggle เป็น Manual override ได้ ตอบโจทย์โรงเรียนที่ยังไม่พร้อม (ใช้ Default) และโรงเรียนที่มีความเฉพาะ (ปรับตามบริบท)

Locked

กคศ. / สพฐ. กำหนด — ผอ. เปลี่ยนไม่ได้

เช่น ชม.สอนขั้นต่ำ 12, ภาระรวม ≥ 20, 8 กลุ่มสาระ
Auto (Default)

ค่า default ของระบบ — ผอ. toggle เป็น Manual ได้

เช่น ชม. PLC 1 ชม./สัปดาห์, หัวหน้าฝ่าย 3 ชม.
Manual

ผอ. กำหนดค่าเอง — ปรับได้ตามบริบทโรงเรียน

เช่น หัวหน้าฝ่ายงบ 4 ชม., PLC 2 ชม.

Universal Config Schema 

{
  key:      "plc_hours_per_week",
  label:    "ชั่วโมง PLC",
  domain:   2,                    // ด้าน 1-4
  default:  1,                    // smart default
  value:    1,                    // actual value in use
  mode:     "auto",               // "auto" | "manual" | "locked"
  lockedBy: null,                 // null | "กคศ." | "สพฐ."
  min:      0,
  max:      5,
  unit:     "ชม./สัปดาห์",
  scope:    "teacher",            // "teacher" | "school" | "system"
}

5. Compliance Alert System

Critical

ด้าน 1 (ชม.สอน) < 12 ชม./สัปดาห์ — ต่ำกว่ามาตรฐาน กคศ.

Warning

ภาระงานรวม < 20 ชม./สัปดาห์ — ต่ำกว่าเกณฑ์

Compliant

ด้าน 1 ≥ 12 ชม. & รวม ≥ 20 ชม. — ผ่านเกณฑ์ กคศ.

6. Unity Strategy — Same Skeleton, Different Lens

ทุกโรลเห็นโครงสร้างเดียวกัน ผ่านเลนส์ต่างกัน — Share component ไม่ใช่ clone page Schedule Grid, Workload Bar, DNA Heatmap, Config Panel ทั้งหมดเป็น shared component ที่แสดงข้อมูลตาม context ของแต่ละโรล

Component นักเรียน ครู ผอ. ผอ.เขต
Schedule Grid ตารางเรียนตัวเอง ตารางสอนตัวเอง ตารางทุกคน + AI Aggregate ทุก ร.ร.
Workload Bar ชม.เรียน ภาระงาน 4 ด้าน ภาระรวมทุกครู ภาระรวมทุก ร.ร.
Config Panel — ไม่เห็น — ไม่เห็น Toggle auto/manual View only
